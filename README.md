# Crusty the Clown - Official Web Page

A fun, single-page HTML site for Crusty the Clown — professional entertainer, balloon artist, and chaos specialist.

## Concept

A colourful, whimsical personal page for Crusty, featuring his bio, services, gallery, and booking info. Pure HTML/CSS/JS — no frameworks, no build tools, just good old-fashioned clowning around on the web.

## Goals

- Capture the chaotic, joyful energy of a professional clown
- Deliver a fast, self-contained single-file experience
- Make it visually playful with CSS animations and bright colours
- Include all key sections: hero, bio, services, gallery, booking form

## Tech Stack

- HTML5
- CSS3 (animations, custom properties, grid/flexbox)
- Vanilla JavaScript (minimal, for interactivity)
- No dependencies, no build step — just open `index.html`
